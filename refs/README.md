# Screenshots folder

Drop your captured reference screenshots here. They'll auto-load into the hover previews and case card images.

## Workflow

1. Open `portfolio.html` in your browser
2. Open DevTools → Console
3. Find the rust-coloured collapsed group **"Portfolio — N screenshots expected in /refs/"**
4. Expand it — it lists every expected filename
5. Capture each one, save here, reload

## Supported formats

`.jpg`, `.png`, `.webp` — checked in that order.

## Recommended dimensions

- Hover-preview screenshots: **640×400px** (will be cropped to 16:10)
- Case card covers: **1200×750px** — filenames `case-a-cover.jpg`, `case-b-cover.jpg`, `case-c-cover.jpg`

If a screenshot isn't found, the page falls back to the colored placeholder block.

## NDA note

When using IA internal screens (PromoSmart, MarkSmart, PriceSmart): blur client names and sensitive data before saving here.
